## 刷题发现的一些有意思的东西
- 字符串List转数组最有效的办法是list.toArray(new String[0])
- 某字符串包含某个字符用str.contains(c + "")
- 字符串去掉某个字符应该用str.replace("#", "")双引号而非单引
- 取余运算(%)效率比较低,特别极端情况下应避免
- 如何创建一维的二维数组: int[][] dest = new int[length][1]
